## Projects Included

### 1. Smart Assistive Blind Stick (Arduino)

An assistive embedded system designed to help visually impaired individuals detect nearby obstacles using ultrasonic sensing.  
Focus areas:
- Sensor integration  
- Real-time obstacle detection  
- Embedded system design  

---

### 2. Ultrasonic Study Timer

A distance-based presence detection system that activates a study timer only when the user is within a defined range.  
Focus areas:
- Ultrasonic sensing  
- Conditional system activation  
- Low-level hardware logic  

---

### 3. CAS-E (Campus-based Social Robot with Emotion) *(Ongoing)*

A collaborative robot (Cobot) with anthropomorphic characteristics focused on wholesome Human-Robot Interaction (HRI).  
Focus areas:
- Emotion-driven interaction  
- Behavioral system modeling  
- AI-assisted response generation  

---

### 4. Robotic Manipulator Mini Project

A foundational implementation of robotic arm modeling and control concepts.  
Focus areas:
- Kinematics fundamentals  
- Control logic implementation  
- Mechanical-to-software integration  

---

### 5. J.A.R.V.I.S – AI Voice Assistant (Python + Gemini API)

A desktop-based AI voice assistant integrating speech recognition, real-time AI response generation, and text-to-speech synthesis.

Key Features:
- Speech-to-Text (Microphone input)
- AI-driven response generation using Gemini API
- Text-to-Speech output
- Multi-threaded conversation handling
- GUI-based interaction using Tkinter

Focus areas:
- Human-AI interaction
- Real-time audio processing
- API integration
- Concurrent system handling (threading)
- Software architecture design

This project demonstrates system-level integration between:
Microphone → Speech Recognition → AI Model → Text Processing → Speech Synthesis → GUI Output
